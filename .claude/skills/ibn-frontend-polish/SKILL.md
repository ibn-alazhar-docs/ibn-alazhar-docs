# Ibn Al-Azhar Docs Frontend Polish Skill

Polish UI after initial generation.

## Must enforce

- Arabic-first copy
- RTL-first layout
- Cairo typography
- #16A34A primary actions
- #CA8A04 restrained heritage accent
- #1F2937 main text
- calm academic spacing
- no generic AI dashboard
- no random gradients
- no excessive glass
- accessible contrast
- clear focus states
- responsive mobile layout
- loading/empty/error/success states

## Output

Return:
- Visual hierarchy fixes
- Typography fixes
- Spacing fixes
- Component fixes
- Accessibility fixes
- RTL fixes
