# Ibn Al-Azhar Docs Brand Audit Skill

Audit all UI and copy against the official Ibn Al-Azhar Docs brand.

## Brand constants

- Product: Ibn Al-Azhar Docs — ابن الأزهر دوكس
- Primary Green: #16A34A
- Heritage Gold: #CA8A04
- Dark Text Gray: #1F2937
- Pure White: #FFFFFF
- Primary Arabic font: Cairo
- Tagline: في بيت كل طالب أزهري
- Brand promise: محتوى تعليمي يليق بتفوق طالب الثانوية الأزهرية

## Required checks

1. Arabic-first.
2. RTL-first.
3. Cairo font is primary.
4. Green is the primary action color.
5. Gold is an accent, not the whole UI.
6. UI feels academic, calm, trustworthy, heritage-modern.
7. Avoid generic SaaS purple/blue gradients.
8. Avoid random glassmorphism.
9. Avoid fake dashboard clutter.
10. Avoid English-first microcopy.
11. Empty/error/success states must sound supportive and professional.

## Output format

Return:
- Pass / Needs work
- Brand violations
- Exact fixes
- Files likely affected
