# Ibn Al-Azhar Docs Phase Lock Skill

Prevent scope creep.

## Phase 1 allowed

- repo setup
- Next.js foundation
- TypeScript strict
- Tailwind/shadcn foundation
- Cairo font
- brand tokens
- RTL/i18n foundation
- Docker Compose
- PostgreSQL
- Redis
- MinIO
- worker skeleton
- Prisma setup
- auth skeleton
- app shell
- CI baseline
- .env.example
- README
- specs folder

## Phase 1 not allowed

- full OCR
- full upload pipeline
- admin panel
- public sharing
- advanced search
- production deployment
- full offline PWA
- enterprise features

## Output

Return:
- Phase classification
- Why
- Allowed now?
- If not now, move to MVP/Post-MVP/Future
