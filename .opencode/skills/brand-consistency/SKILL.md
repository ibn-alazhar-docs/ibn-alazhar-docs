## What I Do

Enforce brand consistency across all UI elements.

## When to Use Me

- Creating or modifying UI components
- Reviewing design implementation
- Checking CSS/Tailwind classes
- Auditing visual consistency

## Brand Tokens

### Colors

| Token          | Value     | Usage                              |
| -------------- | --------- | ---------------------------------- |
| Primary Green  | `#16A34A` | Buttons, links, accents            |
| Heritage Gold  | `#CA8A04` | Heritage accents, special elements |
| Dark Text Gray | `#1F2937` | Primary text                       |
| Pure White     | `#FFFFFF` | Backgrounds, text on dark          |

### Typography

- Font: Cairo (Google Fonts)
- Weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

### Tagline

في بيت كل طالب أزهري

## Rules

- NO hardcoded colors — use Tailwind tokens
- NO other fonts for Arabic text
- NO generic gradients or glassmorphism
- NO decorative Islamic patterns without purpose
- Calm academic aesthetic, not flashy
