## Rationale

The first canonical collection is about **how to seek knowledge itself** — a meta-disciplinary theme that serves every student regardless of their chosen specialization. Topics like the virtue of knowledge, priority fiqh, the instrumental-purposive distinction, gradual learning, etiquette, specialisation balance, authorship, and practical planning form the intellectual infrastructure for all subsequent collections.

This choice grounds the platform in its core mission: to be in every Azhari student's home. Before a student can benefit from collections in fiqh, tafsir, hadith, or Arabic language, they need a framework for seeking knowledge itself.

## Content Design Principles

Each document follows these editorial rules:

1. **Thematic focus:** One clear theme per document, no scope creep
2. **Progressive complexity:** Tiers increase in abstraction and depth
3. **Internal cross-references:** Every doc (except first/last) links forward and backward
4. **Scholarly grounding:** Claims supported by Qur'an, Sunnah, or scholarly tradition
5. **Educational pacing:** Each doc can stand alone but reads better in sequence
6. **Bilingual parity:** AR and EN are adapted, not translated verbatim — natural reading in each language
7. **Callouts used sparingly:** 1–2 per doc maximum, per AUTHORING_TOOLKIT guidelines
8. **Tables used structurally:** Only where comparison adds value
