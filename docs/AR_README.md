## 🎯 ما هو المشروع؟

منصة ويب عربية م Dedicated **لتحويل المستندات الممسوحة ضوئيًا (PDF/صور) إلى نصوص رقمية قابلة للبحث والتحرير** باستخدام تقنيات OCR متقدمة.

**المنتج الحقيقي** هو خط أنابيب تحويل (Pipeline) وليس مجرد موقع قراءة:

```
PDF/صورة  →  OCR (تعرف على النصوص)  →  تنظيف النص العربي  →  Markdown  →  معاينة  →  تصدير
```

الموقع الحالي (Next.js) هو **طبقة العرض والتفاعل** — المستخدم يرفع ملفًا، ويشير التقدم، ويشاهد النتيجة، ويصدرها.

## 🧱 كل الصفحات بالتفصيل

### أولًا: الصفحات العامة (بدون تسجيل دخول)

#### 1. الصفحة الرئيسية `/[locale]`

- **Hero section:** عنوان فرعي + وصف المنصة + زر "ابدأ الآن"
- **قسم الرسالة:** شرح رؤية المنصة
- **مجالات المعرفة:** 4-6 بطاقات لمجالات المعرفة المغطاة
- **Call-to-action:** زر للانتقال للمكتبة

#### 2. مكتبة المستندات `/[locale]/docs`

- **بطاقات الفئات:** عرض فئات المعرفة (مقدمة، تنظيم، طلب العلم...) ببطاقات تفاعلية
- **مسارات القراءة:** بطاقات لمجموعات القراءة المقترحة
- **أحدث المستندات:** قائمة بأحدث 6 مستندات منشورة
- تصميم شبكي (Grid) مع hover effects

#### 3. صفحة فئة `/[locale]/docs/[category]`

- **Header الفئة:** اسم الفئة + وصفها
- **مجموعات موضوعية (Thematic Groups):** تقسيم المستندات داخل الفئة
- **قائمة المستندات:** بطاقات للمستندات مع عنوان ومعاينة

#### 4. عارض المستند `/[locale]/docs/[category]/[slug]`

- **العنوان + البيانات الوصفية:** تاريخ النشر، وقت القراءة، إلخ
- **جدول المحتويات (TOC):** قائمة جانبية بالعناوين في المستند
- **محتوى MDX:** عرض كامل للمستند باستخدام next-mdx-remote
- **Breadcrumbs:** مسار التنقل (الرئيسية > الفئة > المستند)
- **المتطلبات المسبقة:** مستندات يجب قراءتها أولاً بصيغة "قبل أن تبدأ"
- **الروابط المتصلة:** مستندات ذات صلة بعد المحتوى
- **روابط السابق/التالي:** تنقل بين مستندات الفئة

#### 5. مسارات القراءة `/[locale]/journeys`

- **بطاقات المسارات:** قائمة بمسارات القراءة المتاحة مع وصفها

#### 6. مسار قراءة `/[locale]/journeys/[slug]`

- **تفاصيل المسار:** وصف المسار ومستنداته بالترتيب
- **قائمة المستندات:** المستندات المدرجة في المسار

---

### ثانيًا: صفحات لوحة التحكم (Dashboard) — بعد تسجيل الدخول

#### 7. رفع الملفات `/[locale]/files`

- **سحب وإفلات (Drag & Drop):** رفع ملفات PDF/JPG/PNG مباشرة في المتصفح
- **زر رفع تقليدي:** اختيار ملفات من الجهاز
- **التحقق من الصلاحية:** تأكيد نوع الملف وحجمه (حد أقصى 100MB)
- **قائمة الملفات المرفوعة:** عرض مع حالة الرفع
- **show toast:** إشعارات نجاح/فشل الرفع
- **زر "تحويل":** إرسال الملف لخط أنابيب OCR

#### 8. تتبع التحويلات `/[locale]/conversions`

- **جدول التحويلات:** معرف التحويل (Job ID)، اسم الملف، الحالة، تاريخ الإنشاء
- **أشرطة التقدم Progress:** مراحل التحويل (preparing → splitting → ocr → writing → done)
- **تحديث حي (SSE):** التقدم يتغير بدون إعادة تحميل الصفحة
- **أزرار التصدير:** تحميل النتيجة بصيغة TXT أو DOCX أو JSON
- **حالات الخطأ:** عرض رسائل الخطأ وإعادة المحاولة

#### 9. معاينة النتيجة `/[locale]/preview/[id]`

- **عرض Markdown:** معاينة النص المحول بصيغة منسقة
- **شريط أدوات:** تكبير/تصغير، طباعة، نسخ النص
- **معلومات التحويل:** إحصائيات (عدد الصفحات، وقت التحويل، نسبة الثقة)
- **أزرار التصدير:** تنزيل TXT/DOCX/JSON مباشرة من صفحة المعاينة
- **رابط مشاركة:** (مستقبلاً) نسخ رابط عام

#### 10. المجلدات `/[locale]/folders` (قالب — "قريباً")

- **هيكل شجري:** مجلدات هرمية لتنظيم الملفات
- **إجراءات:** إنشاء مجلد، نقل، إعادة تسمية، حذف
- **سلة المحذوفات:** استعادة الملفات المحذوفة

#### 11. الإعدادات `/[locale]/settings` (قالب — "قريباً")

- **DPI:** ضبط دقة OCR
- **التزامن:** إعدادات التخزين السحابي
- **صيغ التصدير:** اختيار الصيغ الافتراضية للتصدير

### رابعًا: واجهات API (5 نقاط)

| المسار                        | الطريقة   | الوظيفة                       |
| ----------------------------- | --------- | ----------------------------- |
| `/api/upload`                 | POST      | رفع ملف PDF/JPG/PNG إلى MinIO |
| `/api/conversion/start`       | POST      | بدء تحويل OCR للملف المرفوع   |
| `/api/conversion/[id]/status` | GET       | الاستعلام عن حالة التحويل     |
| `/api/stream`                 | GET (SSE) | تحديث حي لتقدم التحويل        |
| `/api/export/[id]/[format]`   | GET       | تصدير النتيجة (txt/docx/json) |

## 📌 المشاكل المعروفة

1. **تقطيع لام-ألف** في OCR (لا م ه -> لمه بدلاً من "لم")
2. **أحرف مفردة** تظهر أحيانًا بدون سياق
3. **اكتشاف خاطئ للعناوين** في بعض الحالات
4. **حد الثقة** في OCR يحتاج تحسينًا
5. **اكتشاف القمامة (Garbage)** ليس مثاليًا للنصوص الملوثة
