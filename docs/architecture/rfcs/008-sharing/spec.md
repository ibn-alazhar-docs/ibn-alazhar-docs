# Spec 008: Document Sharing

## Status

- **Status**: approved
- **Phase**: 2D
- **Created**: 2026-06-17
- **Author**: Antigravity

## Overview

ميزة المشاركة تتيح للطلاب الأزهريين إنشاء روابط عامة (Public Links) للمستندات الخاصة بهم بعد اكتمال معالجتها، ليتمكن الزملاء من استعراضها وتحميلها بصيغ (Word, MD, TXT, PDF) دون الحاجة لإنشاء حساب في المنصة. يمكن للمستخدم إلغاء الرابط في أي وقت.

## User Stories

1. **كطالب**، أريد مشاركة ملخص المحاضرة مع زملائي عبر رابط سريع بدلاً من إرسال الملفات بالبريد.
2. **كطالب**، أريد التحكم في مدة صلاحية الرابط (مثلاً 7 أيام) لتجنب انتشار الملفات للأبد.
3. **كمستخدم**، أريد القدرة على إلغاء تفعيل الرابط (Revoke) لقطع الوصول فوراً في حال شاركت الملف بالخطأ.
4. **كزائر (مستقبل الرابط)**، أريد فتح الرابط من هاتفي الجوال واستعراض النص المحسّن وقراءته بوضوح، مع توفر أزرار لتحميله مباشرة لجهازي.

## Acceptance Criteria

1. **إنشاء الرابط**: الزر "مشاركة" يولد رمزاً فريداً (`token`) يخزن في قاعدة البيانات مع معرف المستند والمستخدم.
2. **إدارة الرابط**: نافذة منبثقة (Share Modal) تتيح نسخ الرابط إذا كان موجوداً، أو مسحه (Revoke) إذا أراد المستخدم إيقاف المشاركة.
3. **صفحة العرض العام (`/share/[token]`)**:
   - لا تتطلب مصادقة (No Auth).
   - تقوم بعرض محتوى المستند (Markdown/Text) بتنسيق رائع ويدعم الـ RTL.
   - إذا كان الرابط محذوفاً أو منتهي الصلاحية، تظهر رسالة خطأ أنيقة تشير لعدم صلاحية الرابط.

## Backend Endpoints

- `GET /api/documents/:id/share` - جلب معلومات الرابط الحالي (إن وجد).
- `POST /api/documents/:id/share` - إنشاء رابط جديد للمستند (مع إمكانية تحديد `expiresAt`).
- `DELETE /api/documents/:id/share` - حذف الرابط نهائياً.

## UI/UX Considerations

- استخدام `Framer Motion` لتقديم النافذة المنبثقة بشكل احترافي.
- إضافة زر النسخ للحافظة (Copy to Clipboard) مع رسالة (Toast) لتأكيد النسخ.
- في واجهة الزوار (Public View)، يتم إخفاء أي أزرار تتطلب صلاحيات (مثل التعديل أو الحذف).
